## どうして最新のSAM3じゃなくてSAM2なんだ???
- A.SAM3はトークンの取得が必須だからです(これは例のAviutl2のプラグインも同様です)
- その点、SAM2ならトークンの取得無しで切り抜けるからです
- 弱点としてはテキストプロンプトでの切り抜くもの指定が出来なかったり、
  やや精度が劣ってる、と言ったところでしょうか...。そんな気になるほどでもないと思いますけどね

## 使い方

起動したらまず左上の画像/動画を開くを押して切り抜きたい元の素材を開きましょう。

そしたらプロンプトから前景ポイント/背景ポイント/ボックスのいずれかを選択します。

(マスクしきい値は通常いじらなくても良いです)

そしたらプレビュー画面で○○ポイントを選んでいる場合はクリック、ボックスの場合は範囲選択をして、
切り抜きたい対象を選びましょう！
(前景の場合は選択した物以外が切り抜けて、背景は選択した物を切り抜きます)
- ミスっちゃったら元に戻すボタンを押しましょう(それでさっきの方が良かったらやり直すボタンを押しましょう)
後は出力形式をお好みのものにして出力ボタンを押しましょう...
