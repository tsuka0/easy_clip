## 使い方

起動したらまず左上の画像/動画を開くを押して切り抜きたい元の素材を開きましょう。<br>
そしたらプロンプトから前景ポイント/背景ポイント/ボックスのいずれかを選択します。<br>
(マスクしきい値は通常いじらなくても良いです)<br><br>

そしたらプレビュー画面で○○ポイントを選んでいる場合はクリック、ボックスの場合は範囲選択をして、<br>
切り抜きたい対象を選びましょう！<br>
(前景の場合は選択した物以外が切り抜けて、背景は選択した物を切り抜きます)

- ミスっちゃったら元に戻すボタンを押しましょう(それでさっきの方が良かったらやり直すボタンを押しましょう)

後は出力形式をお好みのものにして出力ボタンを押しましょう...<br>
!!動画ファイルの読み込み、出力等は遅いです!!FPSは24F程で、秒数は長くても10秒くらいにしましょう!!<br>
!!動画での切り抜きはポイントを設置してもそのフレームしか切り取れず、<br>
全てのフレームでその切り抜きたい対象を切り抜く場合は全体に適用ボタンを押してください!!
後、動画の出力GPU版ぶっ壊れてるらしい(これは俺のせいじゃないぞ！！)<br>
最悪画像で頑張ろう！(((<br>

## どうして最新のSAM3じゃなくてefficient-sam2なんだ???

- 更にsam2よりもefficient-sam2の方が2倍ほど処理が早くなるため採用しました
- 弱点としてはテキストプロンプトで切り抜けない事かな
