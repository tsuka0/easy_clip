## _USE SAM2 AI_

簡単に環境を構築出来て、すぐに実行を目標として作成しました。
- zipファイルをダウンロードしましょう(解凍してね!)
- install.batを実行しましょう
- 黒い謎の画面(コマンドプロンプト)が消えるまで待ちましょう。
- 黒い画面が消えたらrun.cmdを実行しましょう！(2回目以降はrun.cmdのみ実行してください)
恐らく起動するはずです！！！

- 起動せずに落ちてしまう人はつっかーへDMで相談してくれたら対応します。
- また、バグもかなりあると思うのでそれも対応します  きっと。

(デザインセンス皆無なのでそこらへん全部AIに書かせました、モダンで良いね！)<br>

## 使い方

起動したらまず左上の画像/動画を開くを押して切り抜きたい元の素材を開きましょう。<br>
そしたらプロンプトから前景ポイント/背景ポイント/ボックスのいずれかを選択します。<br>
(マスクしきい値は通常いじらなくても良いです)<br><br>

そしたらプレビュー画面で○○ポイントを選んでいる場合はクリック、ボックスの場合は範囲選択をして、<br>
切り抜きたい対象を選びましょう！<br>
(前景の場合は選択した物以外が切り抜けて、背景は選択した物を切り抜きます)

- ミスっちゃったら元に戻すボタンを押しましょう(それでさっきの方が良かったらやり直すボタンを押しましょう)<br>
後は出力形式をお好みのものにして出力ボタンを押しましょう...<br>
!注意!動画ファイル読み込みはオマケですが、CPUなので正直使い物にならないくらい時間がかかるよ！<br>
なので動画ファイルを読み込む時は24FPS位の動画かつ最高フルHD程度の10秒程度の動画じゃないと本当にきついよ！<br>

...ごめんじゃん。<br>
